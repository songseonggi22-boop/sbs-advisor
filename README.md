# SBS아카데미 통합 관리 시스템 (Daejeon Branch)

본 프로젝트는 SBS아카데미컴퓨터학원 대전점의 상담 효율화와 마케팅 자동화를 위해 구축되었습니다.

---

## 주요 기능 (Main Features)

### 1. 스마트 패키지 견적 시스템
- 수강생 맞춤형 과정 구성 및 엑셀 견적서 자동 생성
- 5단계 할인 로직 적용 (패키지, 우선등록, 첫결제, 원장권한, 기타)
- 사무실 공용 엑셀 양식(`패키지 견적서(고정).xlsx`) 데이터 자동 바인딩

### 2. 블로그 마케팅 자동 생성기 (SEO Writer)
- 네이버/구글 SEO에 최적화된 블로그 원고 자동 추출
- **홍보 타겟**: SBS아카데미컴퓨터학원 대전점
- **핵심 엔진**: Gemini 1.5 Flash API

---

## 블로그 작성 엄격 지침 (Editing Guidelines)

> **팀장님 전용 7계명 (반드시 준수)**

1. **이모지 절대 금지**: 모든 유니코드 이모지 및 숫자 이모지(①, ② 등) 사용 금지.
2. **모바일 최적화**: 한 문단은 1~3문장, 220자 이내로 짧게 끊어 치기.
3. **가독성**: 문단 사이는 반드시 '빈 줄 1줄'로 구분하여 여백 확보.
4. **구조화**: H1(제목)부터 H4까지 계층 구조를 엄수하여 스캐너블하게 작성.
5. **데이터 시각화**: 본문 내 마크다운 표(Table) 최소 1개 이상 포함.
6. **신뢰성**: 허위/과장 광고 금지, "상담 시 확인" 등 안전 문구 삽입.
7. **전문성**: Q&A 5문항 이상 및 하단 CTA(행동유도) 1개 고정.

## 실행 및 관리 방법

### 1. 로컬 실행 명령어
```powershell

```

- `GEMINI_API_KEY`: Google AI Studio에서 발급받은 키 입력

### 3. 업데이트 및 배포 (GitHub)
```powershell
git add .
git commit -m "기능 업데이트 및 지침 반영"
git push origin master
```
