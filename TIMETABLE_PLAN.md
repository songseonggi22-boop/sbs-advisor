## 2. 데이터 구조 분석

### 2-1. 평일 시트 (`시간표.xlsx` → `평일`)

```
행1: 연도 헤더 (2026년 / 2027년)
행2: 월 헤더  (1월, 2월, ... 12월, 1월, ...)
행3~: 학과 | 시간 | [각 월의 과정명]
```

- 열 = 월 단위 (2026년 1월 ~ 2027년 12월, 총 24개월)
- **날짜 정보 없음** → "몇 월에 수업"까지만 알 수 있음
- 학과: 모션CG, 웹, 인테리어, C강의장, AI, 포트폴리오, 편집, OA, 금특강

### 2-2. 주말 시트 (`시간표.xlsx` → `주말`)

```
행1: 연도 헤더
행2: 월 헤더 (1월, 2월, ...)
행3: 요일 헤더 (토, 일, 토, 일, ...)
행4: 날짜 헤더 (3, 4, 10, 11, 17, 18, ...)  ← 실제 날짜 있음!
행5~: 강의장 | 정원 | 시간 | [각 날짜의 과정명]
```

- 열 = **날짜 단위** (매주 토/일 개별 날짜)
- **"2026년 1월 3일(토)"** 수준의 정밀 정보 제공 가능
- 강의장: A강의장(24명), B강의장(20명), C강의장(15명)

## 3. 기능 설계

### 3-1. 과정명 검색 엔진

```
[검색 입력창]  "에펙" 입력
       ↓
[결과 카드 목록]
 ┌─────────────────────────────────────────┐
 │  에펙                                    │
 │  📍 평일 / 모션CG / 09:00~12:00         │
 │  📅 2026년 1월 (4주 수업)               │
 │                        [+ 장바구니 추가] │
 └─────────────────────────────────────────┘
 ┌─────────────────────────────────────────┐
 │  모션에펙                                │
 │  📍 평일 / 모션CG / 09:00~12:00         │
 │  📅 2026년 2월                          │
 │                        [+ 장바구니 추가] │
 └─────────────────────────────────────────┘
```

**검색 매칭 방식**
- 완전 일치 우선
- 포함 검색 (에펙 → "에펙", "모션에펙", "어드벤스에펙" 모두 검색)
- 대소문자 무시, 공백 무시
- 주말 과정은 `/주말` 구분자로 필터링하여 별도 처리

---

### 3-2. 검색 결과 카드 구조

각 결과 카드에 표시할 정보:

| 필드 | 평일 | 주말 |
|------|------|------|
| 과정명 | 에펙 | 컴활1급 |
| 구분 | 평일 | 주말 |
| 학과/강의장 | 모션CG | B강의장 (20명) |
| 시간대 | 09:00~12:00 | 09:00~12:00 |
| 수업 기간 | 2026년 1월 | 2026년 1월 17일(토) ~ 4월 5일(일) |
| 총 수업 횟수 | - | 12일 (3H×12일) |

---

### 3-3. 장바구니 시스템

```
[장바구니]
┌────────────────────────────────────────────────┐
│ ① 에펙        평일 09시   2026년 1월      [제거] │
│ ② 모션에펙    평일 09시   2026년 2월      [제거] │
│ ③ 컴활1급     주말 09시   1월17일~4월5일  [제거] │
└────────────────────────────────────────────────┘
  ⚠️ ①②는 같은 시간대 — 연속 수강 과정입니다
  
  수강생 이름: [________]
  [📥 시간표 엑셀 생성]
```

**장바구니 기능**
- 과목 추가 / 제거
- 같은 시간대 과목 겹침 감지 → 경고 표시
- 연속 수강 자동 감지 (같은 학과/시간 → "연계 과정"으로 표시)
- session_state로 새로고침해도 유지

---

### 3-4. 일정 시각화

검색된 과목 또는 장바구니 과목들을 **월별 타임라인**으로 시각화:

```
         1월    2월    3월    4월    5월    6월
평일 09시 [에펙][모션에] [어드벤] [블렌더1] [블렌더2] [시포디]
주말 09시 [──────────컴활1급(12일)──────────]
```

- `st.dataframe` 또는 HTML 테이블로 구현
- 같은 행 = 같은 시간대
- 색상 구분: 평일(파랑 계열) / 주말(초록 계열)

## 6. 주말 시트 날짜 파싱 상세

주말 시트 행4에는 실제 날짜(일자)가 있고, 행3에는 요일(토/일), 행2에는 월이 있으므로
3개 행을 조합하면 `(year, month, day, weekday)` 완전한 날짜 생성 가능.

```python
# 예시
col_dates = {
    col_idx: {
        "year": 2026, "month": 1, "day": 3, "weekday": "토"
    },
    ...
}
```

주말 과정 결과 카드에서는:
- **첫 수업일 ~ 마지막 수업일** 표시
- **총 N일 (3H×N일)** 표시 (셀값에서 `3H*N일` 파싱)

## 8. 핵심 데이터 모델

```python
@dataclass
class CourseEntry:
    name: str           # 과정명 (예: "에펙")
    sheet: str          # "평일" or "주말"
    dept: str           # 학과/강의장 (예: "모션CG", "B강의장")
    time: str           # 원본 시간 (예: "9시", "09:00~12:00")
    year: int
    month: int
    day: int | None     # 평일=None, 주말=실제 날짜
    weekday: str | None # 평일=None, 주말="토"/"일"
    info: str           # 추가 정보 (예: "3H*12일")
```

## 9. 고려사항

- **주말 과정 중복 날짜**: 같은 날 같은 강의장에 여러 시간대 과정 → 별도 행으로 표시
- **2026/2027 연도 전환**: 12월→1월 넘어가는 월 헤더 파싱 시 연도 주의
- **Streamlit Cloud 호환**: 템플릿 파일(`★전체시간표 양식.xlsx`)이 로컬 경로라 Cloud에서 접근 불가 → 기본 스타일로 폴백 처리 (이미 구현됨)
- **검색 성능**: `@st.cache_data`로 시간표 파싱 결과 캐싱 필수
